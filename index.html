Smart Home Temperature & Humidity Monitor (Work in Progress)

Developed a device using an STM32F103 microcontroller to monitor environmental conditions via an SHT31 (I2C) sensor. Data is relayed wirelessly using an ESP32 module (via UART AT commands) for display on a mobile app or cloud dashboard.

PCB Design Tool: Altium Designer

Firmware Language: C/C++ (STM32 HAL)

Wireless Protocol: WiFi via ESP32 (using AT commands for MQTT/HTTP)

Debugging Tools: ST-Link Debugger (GDB), UART Serial Debug, Oscilloscope

Skills Demonstrated: STM32 Firmware (HAL), I2C, UART, AT Command Interfacing, PCB Design (Altium), ESP32 Integration

Status: In Progress (Core firmware structure defined)

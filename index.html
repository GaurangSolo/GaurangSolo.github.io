About Me

I'm a dedicated hardware engineer specializing in designing detailed schematics, crafting precise PCB layouts, and developing tailored firmware. Smart Home Temperature and Humidity Monitor

Using the STM32F4, this device monitors temperature and humidity with a DHT22 sensor and sends data to a mobile app via WiFi using an ESP8266.

PCB Design: Integrated STM32F4, DHT22, and ESP8266.

Schematic: Features connections for temperature, humidity sensor, and WiFi module.

Firmware: Utilizes STM32 HAL for sensor interfacing and HTTP requests for cloud data transmission.

IoT-Based Weather Station

This project uses an ESP32 to gather weather data from a BMP180 barometer and CCS811 air quality sensor, sending data to a cloud server via MQTT.

PCB Design: ESP32 with BMP180 and CCS811 sensor integration.

Schematic: Includes sensor connections and voltage level shifting for compatibility.

Firmware: Written in Arduino IDE, implements MQTT protocol for cloud communication.

About Me

I'm a dedicated hardware engineer specializing in designing detailed schematics, crafting precise PCB layouts, and developing tailored firmware. I oversee the entire production process—from sourcing and collaborating with manufacturers in China to navigating regulatory certifications like CE, FCC, and ISED through accredited testing labs.

Additionally, I create comprehensive documentation—including user manuals and assembly guides—to ensure that every product is as user-friendly as it is innovative.

Smart Home Temperature and Humidity Monitor

Using the STM32F4, this device monitors temperature and humidity with a DHT22 sensor and sends data to a mobile app via WiFi using an ESP8266.

PCB Design: Integrated STM32F4, DHT22, and ESP8266.

Schematic: Features connections for sensors and WiFi.

IoT-Based Weather Station

This project uses an ESP32 to gather weather data from a BMP180 barometer and CCS811 air quality sensor, sending data to a cloud server via MQTT.

PCB Design: ESP32 with BMP180 and CCS811 integration.

Schematic: Includes sensor connections and level shifting.

Firmware: Arduino IDE, MQTT protocol for cloud.
